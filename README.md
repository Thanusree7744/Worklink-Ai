
  # WorkLink AI - Frontend

  This is the frontend repository for **WorkLink AI**, a job matching and recruitment platform powered by AI.

  ## Project Structure

  - **Frontend (this repo)**: React + TypeScript + Vite with Tailwind CSS
  - **Backend**: Coming soon (separate repository)

  ## Features

  - Landing page with job search and worker discovery
  - Authentication (Login/Signup for workers and employers)
  - Worker dashboard with profile management
  - Customer/Employer dashboard for job posting
  - AI-powered job recommendations
  - Job details and worker profile pages
  - Admin dashboard
  - Modern UI with Shadcn components

  ## Getting Started

  ### Prerequisites
  - Node.js (v16 or higher)
  - npm, yarn, or pnpm

  ### Installation

  ```bash
  npm install
  # or
  pnpm install
  ```

  ### Development

  ```bash
  npm run dev
  # or
  pnpm dev
  ```

  The development server will start at `http://localhost:5173`

  ### Build

  ```bash
  npm run build
  # or
  pnpm build
  ```

  ## Technology Stack

  - **Framework**: React 18+ with TypeScript
  - **Styling**: Tailwind CSS
  - **Build Tool**: Vite
  - **UI Components**: Shadcn/ui (Radix UI)
  - **State Management**: React hooks

  ## Pages

  - Landing Page - Home page with overview
  - Login/Signup - Authentication pages (Worker & Customer)
  - Worker Dashboard - Personalized worker dashboard
  - Worker Profile - Worker profile and settings
  - Worker Search - Search and discover workers
  - Customer Dashboard - Employer dashboard
  - Job Posting - Create and manage job listings
  - Job Details - Detailed job information
  - AI Recommendations - AI-powered job matches
  - Admin Dashboard - Administrative panel
  - Settings - User settings page
